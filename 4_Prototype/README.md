## –ü—Ä–∏–º–µ—Ä –ö–æ–¥–∞

```python
import copy

class Prototype:
    def clone(self):
        pass

class ConcretePrototype(Prototype):
    def __init__(self, field):
        self.field = field

    def clone(self):
        return copy.deepcopy(self)

# –ò—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ
original = ConcretePrototype("ABC")
clone = original.clone()
```
